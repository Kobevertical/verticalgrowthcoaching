# Vertical Growth Coaching

Website for [verticalgrowthcoaching.com](https://verticalgrowthcoaching.com) — midlife transformation coaching with Jacob V.

## Deployment

This repo auto-deploys to Netlify on every push to `main`.

- **Live site:** https://verticalgrowthcoaching.com
- **Netlify project:** verticalgrowthcoaching

## Structure

- `index.html` — the full single-page site
- `jacob.jpg` — coach photo

## Local preview

Open `index.html` in any browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000
